# Garment Line Monitoring Dashboard

Frontend-only website for a garment line monitoring dashboard, built with Vite and React.

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the local development server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000`

## Production build

```bash
npm run build
npm run preview
```
